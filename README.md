# Executable Governance

> Prototype planner UI exploring how governance rules might be specified as executable artifacts (rather than prose policy documents). Exported from Replit.

## Status

Concept-stage. The UI demonstrates the planning surface; the execution layer is not implemented.

## What to look at instead

For polished, production-grade work see:

- **[Briefcased](https://github.com/sadruzzahan/Briefcased)** — criminal-defense case copilot (multi-agent AI, OCR, PDF, courtroom rehearsal). Public flagship.
- **[Lexor](https://github.com/sadruzzahan/Lexor)** — consumer legal-help SPA with SSE-streamed defense pipeline, Twilio voice/WhatsApp intake, pgvector embeddings.
- **[Beat](https://github.com/sadruzzahan/Beat)** — multi-agent investigation workflows with Mastra, citation verification, chain-of-custody.
- **[Agent-Economy-OS](https://github.com/sadruzzahan/Agent-Economy-OS)** — agents as first-class economic actors with wallets, escrow, reputation, and a runtime API.
